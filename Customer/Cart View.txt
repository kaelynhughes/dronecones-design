Cart View

Picture

View/Feature description (if needed)
   * This page is the cart page where the customer will be able to view what ice cream they have added to their cart, and remove or modify their order.
   * This page will view the order and cost.
   * There will be a checkout button that will take the user to the Checkout View passing it the order information.
   * There will be a delete button that will remove the item out of the users cart.
   * There will be a modify button that will take the user back to the menu so they can change their order.
   * There will be a back button to go back to the menu.
Functions needed
   * Calculate() will take the order and will add up its price to a total.
   * Checkout() will be attached to the checkout button and direct the user to the checkout page including the order.
   * Back() will take the user back to the menu.
   * Delete() will delete the user's order out of the interactive view.
Interactions
   * Pull the price of the products from the inventory table
